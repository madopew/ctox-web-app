Parsing sucks. Predefined algorithm is the answer. ctox will help you to parse a program written in C. The parser itself is built in C, so it's super fast and reliable. Result is written in XML format, so you don't have to worry about using parsed program in your context.
How it works Parsing is the process of analyzing a string of symbols of computer language conforming to the rules of a formal grammar. And CTOX does it in three steps.
1
Lexical analysis
Splits string into separate words called lexemes and combines them with useful information, producing tokens.
2
Syntax analysis
Recursively combines tokens into bigger groups, using some set of rules, in parallel builds abstract syntax tree.
3
Serialization
Converts abstract parsing tree, which is preseneted in memory as call stack, to a readable and easily parseable format XML.
